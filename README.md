# IT Support Specialist Homelab

A hands-on homelab built to develop real IT support skills and create a portfolio of documented lab projects for entry-level IT Support and Help Desk roles.

## About This Lab

This lab simulates a small business IT environment using virtualization. Each project is fully documented with setup steps, troubleshooting scenarios, and outcomes.

## Lab Environment

- Hypervisor: Proxmox VE on dedicated home server
- Domain: lab.local
- Workstations: Windows 10 Pro (domain-joined)
- Server: Windows Server 2022 (Domain Controller, DNS, DHCP)
- Ticketing: osTicket (self-hosted)

## Projects

| Phase | Project | Skills |
|-------|---------|--------|
| 1 | Lab Planning and Architecture | Documentation, network diagramming |
| 2 | Virtualization Setup | Proxmox, VM lifecycle management |
| 3 | Windows Client Support | Windows OS, end-user support |
| 4 | Active Directory Lab | AD DS, DNS, DHCP, Windows Server |
| 5 | User and Group Administration | User accounts, OUs, NTFS permissions |
| 6 | Group Policy Lab | GPO, endpoint configuration |
| 7 | Network Troubleshooting | TCP/IP, DNS, DHCP diagnostics |
| 8 | Remote Support Lab | RDP, remote administration |
| 9 | osTicket Help Desk | ITSM, ticketing workflows, SLA |
| 10 | Software Deployment | MSI deployment, application troubleshooting |
| 11 | Log Analysis and Event Viewer | Windows logs, basic security monitoring |
| 12 | Capstone Support Scenarios | End-to-end IT support methodology |

## Status

Currently in progress. Projects are added as completed.
